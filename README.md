# 📝 Офлайн-заметки с PWA и Push-уведомлениями

## Описание проекта

Веб-приложение для создания заметок и напоминаний, которое:
- Работает офлайн (Service Worker)
- Устанавливается на устройство как PWA (Web App Manifest)
- Использует архитектуру App Shell
- Отправляет push-уведомления через WebSocket
- Поддерживает откладывание напоминаний

## Выполненные практические работы

### №13 — Service Worker
- Регистрация Service Worker
- Кэширование статических ресурсов
- Офлайн-доступ к приложению

### №14 — Web App Manifest
- Файл `manifest.json`
- Иконки разных размеров
- Установка приложения на устройство

### №15 — HTTPS + App Shell
- Локальный HTTPS через `mkcert`
- Архитектура App Shell (каркас + динамический контент)
- Стратегии кэширования: Cache First и Network First

### №16 — WebSocket + Push
- Сервер на Node.js + Express + Socket.IO
- Двусторонняя связь в реальном времени
- Push-уведомления через VAPID-ключи
- Кнопки включения/отключения уведомлений

### №17 — Детализация Push (Напоминания)
- Форма для создания заметок с напоминанием (дата/время)
- Планирование уведомлений на сервере (`setTimeout`)
- Кнопка "Отложить на 5 минут" в уведомлении
- Обработка откладывания через Service Worker

### №18 — Итоговая сдача
- Полностью рабочее приложение
- Открытый репозиторий
- Документация (README.md)

## Технологии

- **Frontend:** HTML, CSS (Chota), JavaScript
- **Backend:** Node.js, Express, Socket.IO
- **Push-уведомления:** web-push, VAPID
- **PWA:** Service Worker, Manifest
- **Хранение:** localStorage
